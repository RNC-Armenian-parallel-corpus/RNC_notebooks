# RNC_notebooks
Этот репозиторий содержит jupyter-блокноты и дополняющие их файлы.
- Блокнот `lingtrain_am_ru.ipynb` + `_exceptions.txt` + `news.zip`. С помощью блокнота можно выровнять парные тексты. Достаточно положить их в папку на уровне блокнота и следовать инструкции по запуску внутри этого блокнота.
Выравнивание осуществляется с помощью библиотеки [lingtrain-aligner](https://github.com/averkij/lingtrain-aligner). Также блокнот доступен в google colab: [Ссылка](https://colab.research.google.com/drive/1A2IhWYKdYOBvRxStCJaeW5Oi3y9-4lhe?usp=sharing). Тестовые файлы `axe_am.txt`, `axe_ru.txt` и `_exceptions.txt` – демонстрационная пара текстов для выравнивания и файл с исключениями соответственно. Файлы `axe.xlsx` и `axe_processed.xml`- примеры результата выравнивания и генерации xml-файла. Чтобы протестировать блокнот на этих файлах, достаточно загрузить их в панель файлов и запустить все ячейки.
- Блокнот для извлечения текстов из XML (на случай, когда нет доступа к исходным выровненным текстам).
- Мини-блокнот с функцией для исправления одного типа ошибок OCR.
- Блокнот для запуска нашей библиотеки по генерации XML (она в отдельном репозитории, там он тоже есть).
